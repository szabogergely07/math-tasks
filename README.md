# Mathe-Abenteuer

Ein deutsches Mathe-Webspiel für Kinder mit:

- Addition und Subtraktion von 0 bis 100
- Multiplikation von 0 bis 10
- Spezialmodus: nur Einmaleins mit 2, 3, 4, 5 und 10
- mehreren Fragetypen
- Punkten, Sternen und Abschlussnote wie in Österreich
- gespeichertem Verlauf für mehrere Kinder

## GitHub Pages aktivieren

1. Repository auf GitHub öffnen.
2. **Settings** öffnen.
3. Links **Pages** wählen.
4. Bei **Build and deployment** unter **Source**: **Deploy from a branch** wählen.
5. Branch **main** und Ordner **/(root)** auswählen.
6. Speichern.

Danach ist die Seite öffentlich erreichbar unter:

`https://szabogergely07.github.io/math-tasks/`

## Dateien

- `index.html` – Struktur der App
- `styles.css` – Design und Layout
- `script.js` – Spiellogik und Punktesystem

## Lokal öffnen

Einfach `index.html` im Browser öffnen.
